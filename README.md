Ellipticoind
==========

Ellipticoind is an Ellipticoin node written in Rust.

Installation:
==========================
1. Clone the repo

  git clone git@github.com:ellipticoin/ellipticoind.git 

2. Install rust

  curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

2. Run `ellipticoind`

  cargo run
